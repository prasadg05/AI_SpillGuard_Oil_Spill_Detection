
# OilGps System

AI-driven oil spill detection system with image upload and PDF report generation.

## Features
Upload satellite images, detect oil spills, calculate confidence scores, record timestamps, estimate affected areas, and generate downloadable PDF reports.

## How to Use
Run the backend with `pip install -r requirements.txt` and `python app.py`, start the frontend with `npm install` and `npm start`, then upload images via the web interface to view results and download reports.
